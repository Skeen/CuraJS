CuraJS
======
The CuraJS is a JavaScript interface for [CuraJS-Engine](https://github.com/Skeen/CuraJS-Engine).
A JavaScript compilation of the [CuraEngine (legacy)](https://github.com/Ultimaker/CuraEngine) by Ultimaker.
